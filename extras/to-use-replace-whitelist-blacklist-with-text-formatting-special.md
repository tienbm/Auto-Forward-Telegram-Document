# 🔦 To Use Replace/Whitelist/Blacklist With Text Formatting Special

### 1. Why Use This Bot?

<mark style="color:red;">Special text formatting can interfere with the functionality of replace, whitelist, and blacklist features. This is because these features rely on plain text matching, and formatting such as bold, italics, links, or emojis may cause mismatches or prevent these features from working as expected. Converting the formatted content into raw text using the bot can help overcome this limitation.</mark>

The [**FindMyIDs**](https://t.me/FindMyIDs_Bot) bot is designed to help users read and extract information from special text formatting in Telegram, including:

* **Bold**
* _Italic_
* `Monospace`
* ~~Strikethrough~~
* [Hyperlinks](https://example.com)
* 🎉 VIP Emojis
* 🅰️ Text Styles

Additionally, the bot supports **converting** these formats into raw content for easy processing and provides tools to utilize features like replace, whitelist, and blacklist efficiently.

### 2. Purpose of Use

The bot helps you:

* View special text formatting in messages.
* Analyze and store information from these formats.
* **Convert text formatting into raw content** for further use or processing.
* Utilize features such as replace, whitelist, and blacklist effectively in Telegram groups and channels.
* Manage content in Telegram groups and channels more efficiently.

### 3. Usage Steps

#### Step 1: Connect to the Bot

1. Search for the @**FindMyIDs\_Bot** on Telegram or access it via this [link](https://t.me/FindMyIDs_Bot).
2. Click **Start** to begin using the bot.

**Step 2: Forward a Message or Post**

<figure><img src="../.gitbook/assets/image (28).png" alt=""><figcaption></figcaption></figure>

* Forward a post or message containing special text formatting directly to the bot.
* Once forwarded, select the **Get HTML** option to receive a detailed output with formatting converted into HTML or raw content.

<figure><img src="../.gitbook/assets/image (29).png" alt=""><figcaption></figcaption></figure>

**Step 3: Receive Results**

The bot will return:

* A list of detected formats.
* Specific content from each format.
* **Raw content** of the message, without formatting.
* <mark style="color:green;">**Use Raw content for replace, whitelist,  blacklist**</mark>
