# Delay For Targets

{% hint style="warning" %}
The **Delay For Targets** feature is designed to prevent Telegram account limitations caused by sending messages to multiple target chats simultaneously when a new message arrives from a source chat. Instead, this feature will delay the forwarding of messages to the targets, spreading out the delivery over time.
{% endhint %}

{% hint style="success" %}
### **Download Mobile App or use Web**

✅ **iOS** → [App Store](https://apps.apple.com/us/app/autoforward-for-telegram/id6447486093)\

{% endhint %}
