You can now organize your forwarding tasks into folders instead of keeping them all in one long list. This makes it easier to:

✅ Categorize by purpose (e.g. Crypto, News, Airdrops…)\
✅ Quickly manage tasks – enable/disable or edit in bulk\
✅ Keep your dashboard clean and intuitive, even with many tasks

👉 Just create a new folder and add your tasks. Managing auto-forward rules is now more organized and efficient than ever!

{% embed url="https://youtu.be/YWzOZKCHHho" %}

#### Toggle All Tasks in Folder

#### 📱 On Mobile:

1. Go to the Folders tab in the app.
2. Long press on the folder you want to manage.
3. An action menu will appear.
4. Select Toggle All Tasks to enable or disable all tasks inside that folder.

***

#### 💻 On Web:

1. Go to the folder management section.
2. Right-click on the folder you want to manage.
3. A context menu will appear.
4. Choose Toggle All Tasks to turn all tasks in the folder on or off.
