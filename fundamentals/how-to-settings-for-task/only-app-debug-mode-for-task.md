# \[Only App] Debug Mode For Task

**DOWNLOAD APP MOBILE NOW**&#x20;

✅ [LINK FOR IOS](https://bit.ly/autoforward-for-telegram-ios)

✅ [LINK FOR ANDROID](https://bit.ly/autoforward-for-telegram-android)

{% hint style="success" %}
Helps resolve issues of not receiving new messages from the chat source.. Only should turn on when you encounter issues
{% endhint %}

{% hint style="danger" %}
**This feature may lead to Telegram limits due to continuous message checking. Do not use it unless there is no other solution.**&#x20;

_**Only enable this feature if you are certain that the task is failing and your configurations are correct**_
{% endhint %}

{% hint style="info" %}
**In some cases when this feature is enabled, events like edit/delete post may not be received.**
{% endhint %}

{% tab title="Instructions" %}
**Follow these steps**

1. Open application **AutoForward for Telegram**
2. At list Task Forward **choose Task you want turn on Debug Mode**
3. At screen detail Task to **Activate Feature**
   1. At block Debug Mode **click&#x20;**<mark style="color:green;">**Turn on**</mark>**&#x20;feature**
   2. **A popup showed select&#x20;**<mark style="color:green;">**ACCEPT**</mark>**&#x20;to activate feature**
   3. <mark style="color:green;">**After successful activation please wait 5 minutes and until there is a new message**</mark>
